# Alpha Dragon Double-Chiral Z-Plane Explorer

An interactive field simulation exploring how structure emerges from the relationship between energy, memory, distance, and chirality.

This build introduces a dual-well system with real-time controls, allowing users to explore how paired structures behave under varying conditions of separation, handedness, and coupling.

---

## 🧠 Core Idea

What if structure is not just driven by energy — but by:

- memory (accumulation)
- distance (interaction scale)
- chirality (handedness)
- coupling (information transfer)

This simulation lets you explore those relationships directly.

- Dual gravity wells (paired systems)

- Bridge formation between wells

- Toroidal transition behavior

---

## 🎮 Controls

### Core Modes
- `1` — Single Well
- `2` — Double Well
- `3` — Toroidal Bias Mode

### Distance (Key Feature)
- Slider — Smooth control of well separation
- `-` / `=` — Adjust separation
- `4` — Tight pair
- `5` — Medium separation
- `6` — Far separation
- `7` — Extreme separation

### Chirality (Handedness)
- `←` — Flip left well chirality
- `→` — Flip right well chirality

### Interaction Controls
- `,` / `.` — Adjust bridge strength
- `[` / `]` — Adjust swirl

### System Toggles
- `Z` — Toggle Z-plane (symmetry split)
- `B` — Toggle bridge overlay
- `H` — Toggle hemisphere overlay

---

## 🔬 What to Look For

### 1. Distance Scaling
Watch how behavior changes as wells move apart:
- Bound → Bridged → Decoupled

### 2. Chirality Interaction
Try:
- Same chirality → stable wrapping
- Opposite chirality → tension, pinch, or instability

### 3. Bridge Formation
At certain distances, particles begin transferring between wells.

### 4. Toroidal Behavior
Close + coherent systems begin to form circular, self-wrapping flows.

### 5. Symmetry Breaking (Z-plane)
Turning on the Z-plane introduces directional bias and hemisphere separation.
